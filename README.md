##  Olá! 😁

Bem-vindo ao meu espaço de projetos!
Sou um **Analista de Sistemas** com interesse especial em explorar e desenvolver soluções voltadas para ambientes Windows. Como parte do meu hobby, dedico tempo à criação de ferramentas de **diagnóstico de sistemas**, buscando melhorias que contribuam para automação de atividades e redução de trabalho.

Além disso, desenvolvo **micro soluções utilizando Java com Spring Boot**, aplicando boas práticas de engenharia de software para construir APIs e serviços que possam ser integrados em diferentes cenários. Meu foco está em criar aplicações enxutas, escaláveis e de fácil manutenção, que sirvam como laboratório de aprendizado e também como referência para outros profissionais.

Este espaço é, portanto, uma vitrine de experimentos e estudos, onde combino **diagnósticos de sistemas, automações, desenvolvimento com Spring Boot e práticas de arquitetura** para criar soluções que procuram unir teoria e prática.
